Sobre mim

Meu primeiro contato com programação veio em 2016 quando ingressei no ensino médio que era integrado ao técnico, no começo a parte front-end não me cativou pois eram uma infinidade de tags para decorar e também não utilizávamos Javascript, ou seja, fazíamos páginas estáticas sem funcionalidades.

Fiquei apaixonado por Java que era uma das matérias, poder escrever códigos para resolver problemas matemáticos mesmo que simples viraram minha paixão. Foi quando percebi que queria que a programação se tornasse minha profissão.

Conclui o ensino médio em 2018 e comecei a aplicar para vagas de programador, devido à baixa experiencia e conhecimentos superficiais me frustrei e comecei a trabalhar na área admistrativa, que foi aonde permaneci até o final de 2020 e fiquei durante todo este período sem qualquer contato com programação.

No começo de 2021 foi quando decidi que voltaria a estudar para conseguir finalmente ingressar na área da tecnologia, mas como havia perdido toda a familiaridade com a programação tive que começar do zero novamente.

Buscando um norte no youtube acabei encontrando um vídeo de um rapaz fazendo um clone do instagram apenas com html e css e fiquei impressionado com o quanto as coisas se completavam. E foi a partir daquele momento que tive a certeza, queria fazer aquilo para o resto da vida.

Hoje conto com alguns projetos no meu GitHub utilizando HTML, CSS e JS. E atualmente estou estudando ReactJS com um projeto em andamento, que também tem um repositório disponível para consulta.
